# 🩺 Diabetes Risk Prediction System

A desktop application developed with **C# Windows Forms** that analyzes diabetes datasets, performs basic data preprocessing, simulates machine learning model training, and predicts diabetes risk for individuals.

> **Academic Project**  
> This project was developed for educational purposes to demonstrate the workflow of a machine learning application, including data preprocessing, model evaluation, and risk prediction.

---

# 📌 Project Overview

Diabetes is one of the most common chronic diseases worldwide. Early identification of high-risk individuals can help prevent severe complications through timely medical intervention.

This application provides a user-friendly desktop interface that allows users to:

- Load diabetes datasets (.csv)
- Perform data preprocessing
- Train multiple machine learning models
- Compare model performances
- Predict diabetes risk for selected individuals
- Display prediction confidence visually

The project focuses on demonstrating the complete machine learning workflow rather than building a production-ready medical system.

---

# 🚀 Features

✅ CSV Dataset Import

- Import diabetes datasets
- Automatic table visualization
- Display dataset information

---

✅ Data Preprocessing

The preprocessing module performs basic cleaning operations including:

- Missing value handling
- Zero-value replacement
- Dataset preparation

---

✅ Machine Learning Models

The application simulates the performance of several popular machine learning algorithms:

- Logistic Regression
- Random Forest
- Support Vector Machine (SVM)
- K-Nearest Neighbors (KNN)

Each model's performance score is displayed for comparison.

---

✅ Diabetes Risk Prediction

Users can select an individual record from the dataset and obtain:

- Diabetes risk percentage
- Risk confidence score
- Visual progress indicator

---

✅ User-Friendly Interface

Developed using Windows Forms with an intuitive graphical interface suitable for beginners and educational demonstrations.

# 📁 Dataset

The project uses the well-known **Pima Indians Diabetes Dataset**, which contains several medical attributes commonly used in diabetes prediction research.

Typical features include:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome
